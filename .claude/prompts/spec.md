# 0. GLOBAL DESIGN SYSTEM

## 🎨 Color System

Primary:
`#7C5DFA` (signature violet)

Supporting:

* `#0B0B0F` → deep charcoal
* `#111118` → section contrast
* `#F5F5FF` → premium off-white
* Subtle gradient accent:
  `linear-gradient(135deg, #7C5DFA 0%, #5A3FD1 100%)`

No excessive glow. Controlled luminosity.

---

## 🔤 Typography

Primary Font: Modern grotesk (Söhne / Geist / Inter Variable refined)

Scale:

* H1: 88–104px
* H2: 48–64px
* H3: 28–32px
* Body: 18px
* Micro: 14px uppercase tracking

Tight line-height for hero.
Generous spacing elsewhere.

---

## ✨ Motion Principles

* Motion = hierarchy.
* No animation without purpose.
* 300–600ms easing cubic-bezier(0.22, 1, 0.36, 1)
* Stagger delay: 80ms incremental.

Inspired by high-end Framer builds.
